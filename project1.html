Project One – Object Orientatd Design

In our fist project, we were tasked with learning the ins and outs of Game Maker. The Core Unit for this project was Unit 71, which revolved around Object Orientated Design for Computer Games.

We had to research in order to understand the main principles and concepts of Object Orientated Design and apply the techniques that we learned through our research into our games design.

In addition to this, we had to research existing games that use Object Orientated Design and apply these to our research. I'm quite fond of RPG Maker games, such as Alice Mare (English translation available here), created using the WOLF RPG Editor and Ib (English available here), created using RPG Maker 2000, which were huge inspirations for me during this project.

In the future, I will explore this type of game design in more depth and to expand on the features that I have previously explored within Game Maker.
